# PTC Cases Directory

This directory contains all Procedure Test Cases (PTCs).

## Organization

- Each PTC should be saved as a separate markdown file
- Use the naming convention: `PTC-YYYY-NNN.md`
- Keep files organized by year or module as needed

## Getting Started

1. Copy `PTC_TEMPLATE.md` from the root directory
2. Rename it following the convention
3. Fill in all sections
4. Save in this directory

## Example

```
ptc_cases/
├── PTC-2025-001.md  (User Registration)
├── PTC-2025-002.md  (User Login)
├── PTC-2025-003.md  (Password Reset)
└── README.md        (This file)
```
